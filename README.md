# User Management Dashboard

A responsive React class-component project for managing users with full CRUD operations using the JSONPlaceholder REST API.

## Features

- View all users in a table
- Real-time search
- Dynamic filters
- Multi-field sorting
- Pagination with page-size control
- Add user form with validation
- Edit user form with pre-filled data
- Delete confirmation prompt
- Friendly error and success messages
- REST API integration using GET, POST, PUT, and DELETE

## Tech Stack

- React
- JavaScript
- CSS
- JSONPlaceholder API

## Run Locally

```bash
npm install
npm start
```

## API

This project uses:
https://jsonplaceholder.typicode.com/users
Note: JSONPlaceholder is a fake API, so add, edit, and delete requests are simulated and are not permanently saved on the server.
